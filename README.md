# DataVisualizer-Node 📊

Uma ferramenta de linha de comando construída para automatizar a criação de relatórios visuais. Este script em **Node.js** lê dados estruturados em um arquivo JSON e gera instantaneamente uma página HTML estilizada com os resultados.

## 🚀 O que este projeto faz?
- Lê e interpreta um arquivo de dados (`dados.json`).
- Processa as informações usando JavaScript no back-end.
- Injeta dinamicamente os dados em um template HTML.
- Gera um arquivo `relatorio.html` pronto para visualização no navegador.

## 🛠️ Tecnologias Utilizadas
- **Node.js** (Módulo genativo `fs` para manipulação de sistema de arquivos)
- **JavaScript** (Lógica de processamento e iteração)
- **HTML5 & CSS3** (Estrutura e estilização do relatório gerado)
- **Linux Terminal** (Ambiente de execução e automação)
